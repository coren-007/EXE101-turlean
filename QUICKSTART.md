# 🚀 Quick Start

## Yêu cầu
- Node.js 18+ hoặc Bun 1.0+

## Chạy dự án

```bash
# 1. Cài dependencies
bun install

# 2. Tạo database + generate Prisma client
bun run db:push

# 3. Seed dữ liệu mẫu (34 môn, 26 gia sư, 12 học sinh)
bun run seed

# 4. Chạy dev server
bun run dev
```

Mở http://localhost:3000
